# AI Topic Space

Interactive AI Topic Space for the Korean AI knowledge ecosystem project.

The site provides a four-level topic hierarchy:

- L0: analytical domain, including Policy, Science, and Technology
- L1: broad thematic family
- L2: intermediate semantic cluster
- L3: fine-grained key phrase node

The homepage opens the semantic zoom browser. Supporting HTML views, data tables, and static snapshots are included for reproducibility.

## Site

After GitHub Pages deployment, the site should be available at:

https://deep1003.github.io/AI_Topic_Space.github.io/

## Main Files

- `index.html`: main semantic zoom browser
- `pages/ai_topic_space_semantic_zoom.html`: full semantic zoom view
- `pages/ai_topic_space_datamapplot_reference.html`: datamapplot-style reference map
- `pages/ai_knowledge_l1_l2_l3_interactive_map.html`: earlier interactive L1/L2/L3 browser
- `data/`: public CSV/JSON metadata used for audit and reproducibility
- `snapshots/`: static PNG figures used in the manuscript and appendix
